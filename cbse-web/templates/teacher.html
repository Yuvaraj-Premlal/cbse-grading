Find ∫ x · eˣ dx using integration by parts. Show all working steps using the ILATE rule. Evaluate the definite integral from 0 to 1.
5
Marks
Medium
Difficulty
10 min
Est. Time
Model Solution
Step 1: Apply ILATE — u = x (Algebraic), dv = eˣdx → du = dx, v = eˣ (1 mark)
Step 2: Apply formula ∫u·dv = uv − ∫v·du → x·eˣ − ∫eˣdx (2 marks)
Step 3: Simplify → xeˣ − eˣ + C = eˣ(x−1) + C (1 mark)
Step 4: Evaluate [eˣ(x−1)]₀¹ = e⁰·0 − e⁰·(−1) = 0 + 1 = 1 (1 mark)
